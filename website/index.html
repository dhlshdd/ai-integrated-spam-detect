Multi-Layered Scam Shield

Our detector evaluates web pages using four advanced defense layers instantly.

🧠

Naive Bayes ML Text Analysis

Scans page text in real-time using natural language processing to identify urgent phrasing, restricted bank alerts, lottery scams, and visual impersonation tactics.

🔍

Visual Clone & Typosquatting

Calculates visual Levenshtein distance on domains to prevent spoofed links like paypa1.com or secure-google.com.

🚨

Form Hijacking Safeguards

Flags cross-origin form actions, insecure HTTP credential inputs, and excessive hidden variables harvesting autocomplete data.

📊

Personal Anomaly Baseline

Establishes a private domain index. Prompts a warning when you visit high-risk layouts or sites you've never browsed before.
